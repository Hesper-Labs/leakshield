|---|---|

| Docs + community | @Hesper-Labs/maintainers |

## Becoming a maintainer

We pull from active contributors. Open a PR, review a few, file a thoughtful
issue or two, and we'll reach out. There's no hidden checklist.

## Reaching us

- Open a GitHub issue for bugs and features.
- Use [GitHub Discussions](https://github.com/Hesper-Labs/leakshield/discussions)
  for questions, design RFCs, or "is anyone else doing X" threads.
- Security disclosures: see [SECURITY.md](SECURITY.md).
