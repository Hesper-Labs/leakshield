# Code of Conduct

This project adopts the
[Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

We pledge to make participation in our community a welcoming experience for
everyone. Read the linked text for the full standards, enforcement
responsibilities, and reporting guidelines.

For incidents, contact the maintainers via the channels listed in
[SECURITY.md](SECURITY.md) and [MAINTAINERS.md](MAINTAINERS.md). All complaints
are reviewed and investigated promptly and fairly.
